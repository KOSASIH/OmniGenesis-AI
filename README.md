## 🚀 Quick Start

### Prerequisites

- Node.js >= 18.x
- Python >= 3.10
- Bun >= 1.3.x (for frontend)
- Git

### Installation

```bash
# Clone the repository
git clone https://github.com/KOSASIH/OmniGenesis-AI.git
cd OmniGenesis-AI

# Install smart contract dependencies
npm install

# Install Python agent framework
cd agents && pip install -r requirements.txt && cd ..

# Install backend
cd backend && pip install -r requirements.txt && cd ..

# Install frontend (Phase 13 dashboard)
cd frontend && bun install && cd ..

# Copy environment config
cp .env.example .env
```

### Compile & Test Contracts

```bash
npx hardhat compile
npx hardhat test

# Test Phase 13 AGI contracts specifically
npx hardhat test test/agi/ConsciousnessToken.test.ts
npx hardhat test test/agi/AGIGovernance.test.ts
npx hardhat test test/agi/SingularityBond.test.ts
```

### Deploy to Testnet

```bash
# Deploy core contracts
npx hardhat run scripts/deploy.ts --network sepolia

# Deploy Phase 13 AGI contracts
npx hardhat run scripts/deploy-phase13.ts --network sepolia
```

### Run Agent Swarm

```bash
python -m agents.core.swarm --config agents/config.yaml --phase 13
```

### Run Backend API

```bash
cd backend && uvicorn app:app --reload
```

### Launch Phase 13 Dashboard

```bash
cd frontend && bun run dev
# Open http://localhost:3000 → navigate to 🧬 AGI Consciousness
```
