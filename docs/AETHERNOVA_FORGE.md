```
   ↑                                                          |

```

## On-Chain Registration

Each innovation gets:
- A unique `contentHash` (SHA-256 of title + description + domain)
- A `metadataURI` pointing to IPFS/Arweave
- Reward in $OMNI based on innovation type multiplier
- 3 validator approvals required
